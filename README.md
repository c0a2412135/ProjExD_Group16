# こうかとん伝説
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* こうかとんをキーボード操作で動かし、敵を倒していくゲーム
* 参考URL：サイトタイトル

## ゲームの遊び方
* wasdキーでこうかとんを操作する
* キーボード操作がないときに, 近くの敵に向かって弾が発射される
* HPが0になるとゲームオーバー


## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画

### 分担追加機能
* 基礎パラメーター（担当：）：HP, ATK, ASなど
* スキル（担当：）：弾の増加, 分散, 貫通, 壁反射
* レベル（担当：）：敵を倒すごとに経験値を得る. レベルが上がると, スキルを一つ追加. プレイヤーのレベルに応じて, 敵のパラメーターも上がる
* 攻撃（担当：）：キーボード操作がないときに, 近くの敵に向かって弾が発射される.
* 敵（担当：）：敵も動き続けるように

### メモ
* クラス内の変数は，すべて，「get_変数名」という名前のメソッドを介してアクセスするように設計してある
* すべてのクラスに関係する関数は，クラスの外で定義してある
